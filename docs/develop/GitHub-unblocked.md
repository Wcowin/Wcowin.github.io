# GitHub 解封指南——This account is flagged

![05f6040d4d8e1c377dc115e8469c7602](https://s1.imagehub.cc/images/2025/12/06/6ff5607333846029ecf71183b3029cef.png)

如果你出现这个提示，恭喜你，说明你的 GitHub 账号被封了。  

但是不要担心，已经被封了两次的我来教你解封。

## 第一步

到 <a href="https://support.github.com" target="_blank">https://support.github.com</a> 提交工单 如果可以提交那记得态度好一点  
当时我写的是：


Hello, Mr. or Ms.
I would like to clarify first, my account is: <a href="https://github.com/Wcowin" target="_blank">https://github.com/Wcowin</a>. I am a college student and I have been using GitHub since my freshman year. Today, I updated my personal blog, and the workflow caused a lag, which was an unintentional action. I thought it was a normal update like usual. I hope you can unblock me.I have used GitHub for four years without any violations. An unintentional workflow submission resulted in my account being flagged, and I am very upset. I hope you can treat me fairly and investigate the truth! Thank you! I hope to receive a reply as soon as possible, thank you!I will continue to maintain the open-source spirit.

---

但是大概率这一步你会卡在需要验证手机号上 那么请看第二步
![image](https://s1.imagehub.cc/images/2025/12/06/18e1afa4bd49a1d70ea7401d98f247d9.png)


## 第二步

需要一个国外的手机号接码，推荐用<a href="https://sms-activate.io/cn/rent" target="_blank">https://sms-activate.io/cn/rent</a> 临时搞个加拿大的手机号接收验证码。  

![image](https://s1.imagehub.cc/images/2025/12/06/1cb3eaf34b1ac50552f19f5af3489c99.png)  

注意：最好是加拿大或者美国的，选择Ins服务那个，然后购买接码就可以。一个号码1.42元人民币。

千万别用什么免费接码平台，免费的接码平台基本上接不到，贪小便宜吃大亏。

## 第三步

同第一步一样，如果还是卡在需要验证手机号上，那么重复第二步。

提交完工单坐等解封即可
