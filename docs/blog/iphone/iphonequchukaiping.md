# 苹果手机如何去除开屏广告
法一：  

操作步骤

1.打开设置

2.找到「屏幕使用时间」并打开

3.找到「内容和隐私访问限制」并打开

4.下滑找到「广告」设置为不允许

完成上面的步骤绝大部分我们常用软件的开屏广告就已经去除了。
