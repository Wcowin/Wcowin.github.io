## 如何加入友链

```html

<div class="card"> 
<img class="ava" src="{avatarurl}" /> 
<div class="card-header"> 
<div> 
<a href="{link}" target="_blank">{name}</a> 
</div> 
<div class="info">{description}</div> 
</div> 
</div>

```

### 示例

```html
    <div class="card"> 
     <img class="ava" src="https://s2.loli.net/2024/02/01/gaE47y5fKM6kosV.png" />

     </div> 
    </div>
```
