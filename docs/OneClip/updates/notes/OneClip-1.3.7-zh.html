🎉 1.3.7版本更新日志

版本 1.3.7

感谢每一位曾经和未来支持 OneClip 的朋友，OneClip 在持续更新中。

我们的QQ群号请加一下：1060157293 ｜【点我加入 OneClip 交流群】

功能修复

  • 修复了提bug的用户
  • 修复菜单栏右键粘贴不起作用问题
  • 修复自定义存储位置无法切换问题
  • 修复右键菜单重复的问题
  • 修复堆栈功能不能连续使用的问题，修复堆栈图片无法粘贴问题
  • 修复菜单栏浏览分类无法悬浮预览问题
  • 修复大量数据批量操作造成卡死问题
  • 修复卡片模式标题位置错误问题

功能新增

  • 新增 SQLite 存储架构，这是 OneClip 里程碑式的升级改进
  • 新增复制屏蔽图片、文件等等选项
  • 支持历史记录拖拽到分类标签、收藏、快捷回复
  • 新增从旧的架构迁移到新架构的测试功能

功能改进

  • 改进右键菜单
  • 进一步结合 SQLite 来改进去重功能
