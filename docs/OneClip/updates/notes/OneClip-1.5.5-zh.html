🎉 1.5.5版本更新日志

版本 1.5.5
感谢每一位支持OneClip的朋友，OneClip在持续更新中。
欢迎加入我们的社区：OneClip 交流群 | TG 群组

功能修复

  • 修复raw_pasteboard_data过滤逻辑错误
  • 修复数据库占用过大的问题
  • 修复苹果通用同步后 OneClip 错误捕获为rtfd文件的问题（感谢QQ群热心用户👻）
  • 菜单栏 popover 小概率无法正常滚动到第一个历史记录的问题
  • 密码保护功能的漏洞
  • 关闭"悬浮预览"开关后行的悬浮高亮效果无法正常工作问题
  • 文本的 trim 操作去掉

功能新增

  • 右键菜单新增文本选词功能
  • 静默OCR/翻译功能支持二次查看编辑，所有静默操作统一行为：带"查看结果"按钮 HUD，2秒自动消失
  • 主窗口：支持 Ctrl+P/N 上下导航；快速粘贴面板：支持 Ctrl+P/N/F/B 上/下选择 + 前/后翻页
  • 菜单栏图标左键新增"显示菜单"，也就是说可以让左键和右键效果一样
  • 设置-快捷键里新增自定义拆分多行入栈快捷键⌘+⇧+S
  • AI结果窗口添加ESC键关闭功能
  • 脚本引擎支持网络请求，功能更加强大了。新增了 JavaScript 调用原生 API 的机制
  • 菜单栏和快速粘贴面板新增标题显示，另外菜单栏还补全了Cmd+Z 撤销快捷键
  • 新增开发者对大家的💛

功能改进

  • 改进钉图窗口的关闭逻辑
  • 快速粘贴面板固定后按键不再放行，默认①-⑨执行粘贴功能、⌘+1-9执行复制功能
  • 特别提醒：考虑到先前Mac云同步暴露的诸多问题，这个版本重新设计重写此功能，现在的Mac同步是基于局域网同步
  • 改进密码保护设计
  • 快速粘贴面板的一些交互改进
